CSGO Lounge Spam Blocking

Blocking unwanted trade, gambling and phising advertising comments on CSGO Lounge

Posted by Gergely Szabo on April 11, 2018

Intro

I have grown tired of clicking “remove” and “block” on dozens, possible hundreds of spam replies on my Lounge trades every few days so I wrote a script that does just that. It’s written in JavaScript so it runs in the browser, I have tested it in Chrome, but there is not reason it should not work in others like Firefox.

How does it work?

When you go to one of your trades the script checks the replies and if it finds texts like “Go trade cṣgogem.com” it deletes the reply and blocks the sender so they will not be able to comment any more. If you find it does not block some of the spam or falsely blocks it then please contact me and I will update the script. Setup information is presented after the code.


Setup


Chrome

  • Install the tampermonkey extension from the chrome web store.
  • Click the extension’s icon then “Create a new script”. Paste in the code from above and save it.
  • You are set up.

Firefox

  • Add the Greasemonkey add-on to your browser.
  • Click on the monkey head then new script and paste in the code from above.
  • Save it and you are set up.

Feel free to comment if you have any questions or request.
